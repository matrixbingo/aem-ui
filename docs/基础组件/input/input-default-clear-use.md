# input-default-clear

`简化 Input，返回 string, 如果 disabled 为 ture，则清空 Form 对应数据`


## 代码演示

### 基础用法

<code src="./input-default-clear-use.tsx" />


## API

```jsx | pure
<InputDefaultClear value={value} onChange={onChange} disabled={disabled} />
```


### Params
| 参数     | 说明                   | 类型        | 默认值 | 必填 |
| -------- | ---------------------- | ----------- | ------ | ---- |
| value    | 输入内容               | string      | ''     | 否   |
| onChange | 输入框内容变化时的回调 | function(e) | ''     | 否   |
