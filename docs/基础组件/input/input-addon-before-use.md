## API

```jsx | pure
<InputAddonBefore value={value} addonBefore={addonBefore} onChange={onChange} />
```


### Params
| 参数             | 说明                   | 类型        | 默认值 | 必填 |
| ---------------- | ---------------------- | ----------- | ------ | ---- |
| value            | 输入内容               | string      | ''     | 否   |
| onChange         | 输入框内容变化时的回调 | function(e) | ''     | 否   |
| addonBeforeValue | 前半部分               | string      | ''     | 否   |
