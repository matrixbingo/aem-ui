## API
```jsx | pure
 <Label title="title" level={2} />
```

### Params

| 参数  | 说明               | 类型   | 默认值 | 必填 |
| ----- | ------------------ | ------ | ------ | ---- |
| title | 标签字体，组合使用 | string | ''     | 是   |
| level | h2,h3....          | number | 2      | 否   |
