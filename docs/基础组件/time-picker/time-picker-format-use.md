## API
```jsx | pure
<TimePickerFormat onChange={onChange} />
```


### Params
| 参数     | 说明                   | 类型        | 默认值     |
| -------- | ---------------------- | ----------- | ---------- |
| value    | 时间内容               | string      | -          |
| onChange | 输入框内容变化时的回调 | function(e) | -          |
| format   | 时间格式化             | string      | 'HH:mm:ss' |

其他 TimePickerProps https://ant.design/components/time-picker-cn/#API
