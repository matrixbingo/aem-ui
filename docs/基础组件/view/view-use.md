## API


### Params

| 参数    | 说明                | 类型                | 默认值 | 必填 |
| ------- | ------------------- | ------------------- | ------ | ---- |
| visible | 可见与否            | boolean             | true   | 否   |
| destroy | 销毁与否            | boolean             | false  | 否   |
| space   | 是否使用  space     | boolean             | false  | 否   |
| style   | React.CSSProperties | React.CSSProperties | -      | 否   |
